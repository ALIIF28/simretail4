# SimRetail POS

**SimRetail POS** adalah sebuah aplikasi web simulasi manajemen operasional retail (toko). Proyek ini dirancang sebagai sistem informasi retail sederhana yang mencakup berbagai fungsi operasional mulai dari manajemen gudang hingga transaksi di kasir.

## âœ¨ Fitur Utama

Aplikasi ini memiliki beberapa modul utama berdasarkan peran pengguna:

* **Dashboard Admin**: Mengelola data master dan konfigurasi sistem.
* **Manajemen Gudang (Inventory)**: Memantau stok barang, input barang masuk, dan distribusi.
* **Sistem Kasir (Point of Sale)**: Antarmuka transaksi penjualan barang secara real-time.
* **Panel Owner**: Laporan ringkas untuk memantau performa toko secara keseluruhan.
* **Dark Mode Support**: Antarmuka yang nyaman untuk mata dengan fitur mode gelap.
